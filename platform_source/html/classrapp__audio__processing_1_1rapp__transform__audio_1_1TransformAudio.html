Detailed Description

Provides audio type tranformation functionalities.

Allows the transformation of an audio file to a different format. Supports the alteration of the type (i.e. headset, nao_wav_1_ch etc.), the sample rate, the audio channel number, the audio format and the audio name. Handles transform audio service callback (rapp_audio_processing.rapp_audio_processing.AudioProcessing::transform_audio)
